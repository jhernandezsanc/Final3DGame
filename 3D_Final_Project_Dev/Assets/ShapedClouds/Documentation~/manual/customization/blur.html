Blur

Scripting API

Settings for bidirectional blur.

Inspector Settings

It doesn't have many options to worry about:

When disabled, it has no performance impact.

Sample Count

Determines how many samples the blur effect should take. Does not include the middle sample, and is sampled in each direction.

A number of 1 would end up with 3 samples total for the first pass of the blur, the middle one, and 2 for the sides. This ends up with a total of 6 samples, and that's the lowest possible setting. Careful when using this blur with large resolution textures, it's meant to hide low resolutions and not make high resolutions softer.

Intensity

Distance in UV space the blur goes, with the horizontal pass adjusted to be the same distance as the vertical one. Samples are evenly spread across this distance.

Shader

You can set a custom shader for the blur effect.

It's better to make a copy of the original and make changes/improvements, rather than creating a new one from scratch.

When To Use?

The blur effect is particularly useful when rendering sharp edges and meshes to the Base Texture, or smoothing low resolution sharp edges on the Final Texture.

## 2. Central Concept

**Description:**  
A 3D FPS platformer inspired by Doom, Cyberpunk, Quake and other action games. Player progresses through levels using shotgun recoil to direct trajectory mid-air, moving from platform to platform, and destorying enemies to clear levels. 

---

## 4. Game Design

### Player Controls

| Action              | Input |
|--------------------|------|
| Move               | W / A / S / D |
| Jump               | Space Bar *(Gravity: -20 u/s², Jump Height: 3 units)* |
| Attack / Interact  | Left Mouse Button |
| Camera             | Mouse *(First-Person Camera — PlayerScripts/MouseMovement.cs)* |
| Pause / Menu       | |

---

### Win / Lose Conditions

- **Lose:** Player falls off the level  
- **Win:** Player completes the level
